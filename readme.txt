== Description ==

Travel Stories is a fresh and clean theme for hotel accommodations, travel bloggers, travel agencies and other travel- and tourism-related websites. The theme includes an extensive documentation and is really easy to use. Travel Stories is perfect for people who need a ready-to-go website for their hotel or travel agency. Also, it is highly customizable to go with your marketing plan or to match your brand. Travel Stories will be awesome for Travel Diaries, Tourist Agencies, Travel Magazines, Tourist Destinations, Photography, Handmade and DIY Workshops, Travel Storiess, Culture Overviews and Tutorials, Event Information, Cuisine, etc. Live demo at http://wp-demo-theme.bestweblayout.com/travelstories/

== Installation ==

Manual installation:

1. Upload the Travel Stories folder to `/wp-content/themes/` directory

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'travelStories'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activation and Use

1. Activate the Theme through the 'Themes' menu in WordPress

== Theme Notes ==

= Slider =

Once the theme is installed, the metabox "Screen Options" will appear in the Wordpress admin panel in the post editing section.

This metabox features a Slider checkbox. When checking it, another checkbox 'Display this post in the Slider' will appear at the bottom of the screen. Each post featured image you marked the aforesaid checkbox for will be displayed in the slider. We recommend using images no smaller than 1600*800px. You can add as many posts to the slider as you wish, yet the slider should feature a minimum of one post. 

= Featured Post =

Once the theme is installed, the metabox "Screen Options" will appear in the Wordpress admin panel in the post editing section.

This metabox features a 'Featured block' checkbox. When checking it, another checkbox 'Display this post in the Featured block' will appear at the bottom of the screen. Each post featured image you marked the aforesaid checkbox for will be displayed in the Featured block. We recommend using images no smaller than 1100*500px. The theme must include a Featured Post. Only one post can be marked as Featured Post.

= Post Thumbnail Functionality =

For post thumbnails, we suggest using images with the width equal to 1600 px. The recommended number of displayed posts is 9 or any other number that is multiple of three. This option can be set in the Dashboard WordPress - Setttings - Reading. Please set the number of posts displayed at once on the home page in 'Blog Pages show at most' field. 

= Widgets =

The Theme includes a standart Widget and the Footer widget.

== Frequently Asked Questions ==

= How can I change the language from English into Russian? =

Change the language in the admin settings screen. Settings > General > Site Language.

= Does the theme support widgets? =

Yes, this theme includes widgets. The following widgets are shown by default: recent posts, recent comments, archives, categories.

= How can I get the slider working, where do I install it? =

In this theme, the slider is installed by default. To have it displayed, please add at least one post to the slider (using the metabox in edit post mode in Wordpress admin panel). To remove the slider, please unmark all the checkboxes you marked for all the posts to be added to the slider.
